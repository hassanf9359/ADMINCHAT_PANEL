## йҖҡдҝЎжөҒзЁӢ

### з”ЁжҲ·з§ҒиҒҠ Bot вҶ’ Web йқўжқҝ

```
TG User вҶ’ Bot(aiogram) вҶ’ еӯҳDB + Redis pub вҶ’ WebSocket вҶ’ Web йқўжқҝ
```

### Web йқўжқҝеӣһеӨҚ вҶ’ TG з”ЁжҲ·

```
Web йқўжқҝ вҶ’ REST API вҶ’ Bot Dispatcher(йҖүbot) вҶ’ aiogram send вҶ’ TG User
```

### зҫӨз»„ @Bot вҶ’ Web вҶ’ еӣһеӨҚ

```
TGзҫӨ@Bot вҶ’ Bot Handler вҶ’ еӯҳDB + Redis pub вҶ’ WebSocket вҶ’ Webйқўжқҝ
WebеӣһеӨҚ вҶ’ Dispatcher вҶ’ дјҳе…ҲеҺҹBot вҶ’ 429? вҶ’ жҚўBotжұ е…¶д»–Bot вҶ’ replyз”ЁжҲ·
```
