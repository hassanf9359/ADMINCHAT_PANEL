## 🚀 What this app does

ADMINCHAT_PANEL is a desktop-ready support panel for Telegram customer service. It helps one team handle many chats in one place. You can read messages, reply to users, and manage support work from a single screen.

It is built for Windows users who want a simple way to run a Telegram support desk. It works with a multi-bot setup, so one panel can manage more than one bot. It also includes FAQ tools, live web chat, user management, and AI help for common questions.

## 🖥️ Before you start

Use a Windows PC with a stable internet connection. For smooth use, keep at least 4 GB of RAM free and make sure you have enough disk space for the app and its data.

You will also need:
- A Telegram bot token for each bot you want to connect
- Access to the admin panel settings
- A modern web browser such as Edge, Chrome, or Firefox

If your team uses a local server, keep the app and the server on the same network.

## 💬 Main features

ADMINCHAT_PANEL gives you a single place to manage support work.

### 🔄 Multi-bot pool
Connect more than one Telegram bot at the same time. This helps split support work across teams or brands.

### 💌 Two-way messaging
Read user messages and reply from the same panel. You do not need to switch between tools.

### 📚 FAQ engine
Set up answers for common questions. The FAQ system can work in several modes, including rule-based responses and AI-assisted replies.

### 🌐 Real-time web chat
Use live chat in the browser to answer users fast. Messages update at once, so agents see new chats without refresh.

### 👥 User management
View user records, update support status, and keep track of customer history.

### 🧠 AI support mode
Use AI for common questions and quick draft replies. You can keep control over the final response.

## 🧭 How to use it

1. Open ADMINCHAT_PANEL.
2. Choose the bot or inbox you want to manage.
3. Read the new message list.
4. Click a chat to open the conversation.
5. Type your reply in the message box.
6. Send the reply.
7. Mark the chat as solved when done.

You can also use the FAQ tools to answer repeated questions with less typing.

## 🔐 Telegram bot setup

To connect a bot:
1. Create or open your bot in Telegram.
2. Copy the bot token.
3. Paste the token into the app settings.
4. Save the token.
5. Test the connection.

Repeat the same steps for each bot in your pool.

If you use several bots, give each one a clear name so your team can tell them apart.

## 🧩 FAQ modes

The app supports different FAQ modes so you can match your support flow.

Common modes include:
- Fixed reply mode for direct answers
- Keyword match mode for question-based routing
- Category mode for grouped help topics
- AI mode for smart answer suggestions
- Hybrid mode for mixed manual and AI support
- Auto-reply mode for first response handling
- Agent assist mode for draft replies
- Escalation mode for cases that need human review

Use a mode that fits your support team size and message volume.

## 🧰 Troubleshooting

If the app does not open:
- Make sure the download finished
- Try running it again
- Check that Windows did not block the file
- Restart your PC and try once more

If a bot does not connect:
- Check the token
- Make sure the bot is active in Telegram
- Confirm the internet connection
- Save the settings again

If messages do not appear in real time:
- Refresh the app
- Check the server or local network
- Confirm the websocket connection is active

If FAQ replies do not trigger:
- Check the trigger word
- Make sure the mode is correct
- Save the rule again
