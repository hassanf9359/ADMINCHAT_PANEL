# ADMINCHAT Panel - 前端页面与路由设计

## 技术栈

- React 18 + TypeScript
- Vite (构建)
- React Router v6 (路由)
- Zustand (状态管理)
- shadcn/ui + Tailwind CSS (UI)
- Lucide Icons (图标)
- TanStack Query (React Query) (数据获取/缓存)
- react-markdown / remark (Markdown 渲染)

## 路由表

| 路径 | 页面组件 | 说明 | 最低权限 |
|------|---------|------|----------|
| `/login` | Login | 登录页 | 公开 |
| `/` | Dashboard | 统计面板 (首页) | agent |
| `/chat` | Chat | 聊天主页 | agent |
| `/chat/:conversationId` | Chat | 聊天 (选中某会话) | agent |
| `/users` | UsersGrid | 用户网格 | agent |
| `/users/:id` | UserDetail | 用户详情 | agent |
| `/blacklist` | Blacklist | 黑名单列表 | agent |
| `/bots` | BotPool | Bot 池管理 | admin |
| `/faq` | FAQList | FAQ 规则列表 | admin |
| `/faq/new` | FAQEditor | 新增 FAQ | admin |
| `/faq/:id/edit` | FAQEditor | 编辑 FAQ | admin |
| `/faq/ranking` | FAQRanking | FAQ 命中排行 | agent |
| `/faq/missed` | MissedKnowledge | 遗漏知识点 | admin |
| `/requests` | MovieRequests | 求片管理 | admin |
| `/ai` | AISettings | AI 配置 | super_admin |
| `/admins` | AdminManage | 管理员管理 | super_admin |
| `/settings` | Settings | 系统设置 | super_admin |

**侧边栏会话列表规则:**
- 有新消息的排最上面
- 🔴 未处理 / 🟢 已处理 状态标识
- 群消息显示: `群{群名} - {用户名}` (下排 TGUID)
- 私聊显示: `{用户名}` (下排 TGUID)
- 显示用户 tag 标签
- 支持按状态筛选、搜索

**聊天窗口:**
- 消息气泡区分方向 (用户在左，管理员在右)
- 显示发送 bot 名称、FAQ 匹配标记
- 图片/视频/文件可直接预览
- Markdown 正确渲染
- 支持引用回复

**输入框:**
- Markdown 编辑 (可切换预览)
- 图片上传 (拖拽/粘贴/按钮)
- 发送快捷键 Ctrl+Enter

### 5. UserDetail 用户详情

- 完整 TG 信息展示 (头像、用户名、TGUID、DC、地区、Premium 等)
- 标签管理 (添加/删除)
- 分组管理
- 拉黑/取消拉黑 按钮
- 历史会话列表 (点击可跳转到 Chat 页面)
- Turnstile 验证状态

### 7. FAQList FAQ 列表

- 表格展示: 规则名、关联问题数、关联答案数、匹配模式、回复模式、命中次数、状态
- 操作: 编辑、启用/禁用、删除
- 筛选: 按回复模式、状态

### 8. FAQEditor FAQ 编辑器

见 FAQ_ENGINE.md 中的 UI 设计。

左右分屏，蓝色问题列表 + 绿色答案列表，勾选关联。

### 9. FAQRanking 问题排行榜

- 柱状图 + 表格
- 排序: 命中次数降序
- 时间筛选: 今日/本周/本月/全部

### 10. MissedKnowledge 遗漏知识点

- 表格: 关键词、出现次数、示例消息、最近出现时间
- 操作: [创建FAQ] → 跳转 FAQEditor 预填关键词、[删除]
- 上次更新时间 显示

### 11. AISettings AI 配置

- AI 服务商配置 (base_url, api_key, model)
- 默认参数 (temperature, max_tokens 等)
- 测试连接按钮
- 用量统计图表

### 12. AdminManage 管理员管理

- 表格: 用户名、角色、状态、创建时间
- 操作: 新增、编辑角色/权限、停用
- 权限矩阵编辑 (勾选各权限)

### 13. Settings 系统设置

- Turnstile 配置 (secret key, 有效天数)
- 媒体缓存天数
- 会话分配开关
- 遗漏知识点更新时间
- 其他全局参数

### 关键实时事件

| 事件 | 触发 | UI 更新 |
|------|------|---------|
| `new_message` | 新 TG 消息到达 | 聊天窗口追加气泡 + 侧边栏排序 |
| `conversation_updated` | 会话状态变化 | 侧边栏状态标识更新 |
| `bot_status` | Bot 限流/恢复 | Dashboard + BotPool 状态 |
| `stats_update` | 统计数据变化 | Dashboard 数字更新 |
